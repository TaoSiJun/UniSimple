# UniSimple
> A simple game framework for Unity3D.

## Core
基础的一些模块

- BindableProperty 数据绑定
- TypeStore 类型存储管理
- EventCenter 事件中心
- IUpdatable 可更新接口
- SimplePlayerPrefs 简单的本地缓存

## GameState
游戏状态管理器

## Machine
分层状态机

## Localization
本地化系统

## Pool
对象池

- C# 类
- GameObject 游戏对象

## Singleton
单例系统

## YooAsset
**YooAsset** 资源管理方案相关封装

- YooAssetManager: 接口管理和引用计数
- AtlasManager: 管理 Unity Sprite Atlas
